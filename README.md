# 標案批次查詢工具

這個資料夾內的工具可以批次查詢政府電子採購網：

https://web.pcc.gov.tw/prkms/tender/common/basic/indexTenderBasic

## 開啟 UI 畫面

直接雙擊：

```powershell
run_ui.bat
```

或在 PowerShell 執行：

```powershell
cd D:\工具開發\tender-notice
.\run_ui.bat
```

UI 分成三個分頁：

- **招標查詢**：查詢政府電子採購網的招標公告
- **公開徵求查詢**：查詢公開徵求（`readTpAppeal`）
- **關鍵字管理**：新增/刪除分類與標案名稱，兩個查詢分頁共用同一份清單

共同功能：

- 可用 `keywords.json` 建立分類與標案名稱
- 可匯入/匯出關鍵字分類 JSON
- 分類可單選或多選，查詢時會自動帶入該分類底下的所有標案名稱
- 可手動輸入臨時標案名稱
- 匯出 Excel `.xlsx`，中文表頭，連結可直接點選

招標查詢分頁：

- 查詢「當日」、「等標期內」、「日期區間」
- 預設使用「日期區間」並自動帶入最近 3 天，日期以西元格式送出
- 表格列出機關、案號、標案名稱、公告日期、截止投標、預算金額
- 選取資料後可開啟標案頁面

公開徵求查詢分頁：

- 查詢條件為標案名稱與公開徵求日期（起訖），預設帶入最近 3 個月
- 日期區間請勿超過 3 個月（政府採購網限制）
- 不選分類、也不輸入標案名稱時，會查詢區間內全部公開徵求
- 表格列出機關、案號、標案名稱、公告日期、截止日期、公告次數
- 選取資料後可開啟徵求頁面

## 第一次使用

`run_ui.bat` 會自動安裝必要套件：

```powershell
py -m pip install -r requirements.txt
```

## 關鍵字分類

`keywords.json` 格式如下：

```json
{
  "categories": {
    "土地地政": ["土地", "地政"],
    "資訊維護": ["資訊", "維護"]
  }
}
```

在 UI 裡也可以新增/刪除分類，或把關鍵字加入選取的分類。

新增分類內關鍵字的流程：

1. 切到「關鍵字管理」分頁
2. 在左邊「分類」清單點選一個分類
3. 按「新增關鍵字到選取分類」
4. 輸入關鍵字，可用逗號、頓號、空白或換行分隔
5. 程式會自動寫回 `keywords.json`

查詢時只要選取分類，就會查該分類底下所有關鍵字；不用再把右邊關鍵字反藍。右邊反藍主要是用來刪除關鍵字，或在沒有選分類時單獨挑幾個關鍵字查詢。

「關鍵字管理」分頁的分類一次只能選一個；新增/刪除關鍵字只會修改目前選取的分類。

匯入/匯出格式：

```json
{
  "categories": {
    "土地地政": ["土地", "地政"],
    "資訊維護": ["資訊", "維護"]
  }
}
```

## 命令列備用

也可以不用 UI，直接用命令列查詢：

```powershell
py tender_search.py 資訊 維護 系統
```

查詢等標期內：

```powershell
py tender_search.py 資訊 維護 --date-type period
```

查詢日期區間：

```powershell
py tender_search.py 資訊 --date-type range --start-date 115/07/01 --end-date 115/07/08
```

日期區間也可以輸入西元年，例如：

```powershell
py tender_search.py 土地 --date-type range --start-date 2026/07/06 --end-date 2026/07/08
```

註：政府採購網的 SSL 憑證在部分 Python 版本會驗證失敗，因此工具預設不啟用憑證驗證。
